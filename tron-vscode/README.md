# T.R.O.N. Developer Integration

The official VS Code extension for the T.R.O.N. Task Routing & Orchestration Node.

## Features

* **Native Task Management:** View your Basecamp, Jira, or Monday tasks directly inside the VS Code sidebar.
* **One-Click Workflow:** Click any task to instantly transition its state and check out the corresponding feature branch locally.
* **Quick Add:** Brainstorm and add new tickets directly to your PM board's backlog without leaving the editor.

## Requirements

This extension requires the T.R.O.N. Node.js API Gateway to be running (either locally via Docker or hosted in the cloud). 

## Extension Settings

Ensure your backend API URL is correctly mapped in the extension source before packaging.
